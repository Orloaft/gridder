# Reward Reveal System - Grid-Based Implementation

## Design Philosophy
All reward reveal animations are represented using the existing uniform grid cell system. Every visual element (gold counters, item slots, particles, etc.) exists as grid tiles, maintaining visual consistency with the rest of the game while delivering maximum juice and satisfaction.

## Phase-by-Phase Breakdown

### Phase 0: Victory Transition (1 second)

**Grid State:**
- Entire grid fills with victory color (gold gradient)
- Center cells pulse outward in wave pattern
- Text tiles appear: "VICTORY!" or "WAVE [X] COMPLETE!"

**Grid Animation:**
```
Frame 1: All cells flash white
Frame 2-10: Gold gradient ripples from center outward
Frame 11-20: "VICTORY" tiles fade in at rows 7-9
```

**Audio:** Triumphant fanfare, screen shake on initial flash

### Slot Machine Sequencing

**Slots reveal left-to-right with stagger:**
- Slot 1 starts spinning immediately
- Slot 2 starts 0.2s after Slot 1
- Slot 3 starts 0.2s after Slot 2

**Stopping sequence:**
- Each slot takes 1-2 seconds from spin start to reveal
- Legendary items always reveal LAST for maximum impact

## Skip/Fast-Forward System

**User Control:**

**Tap/Click:**
- Speeds up current animation phase to 2x speed
- Gold/gem counters jump to final value immediately
- Slot spins stop within 0.5s

**Hold Button:**
- After 2 seconds of holding, skip to final results
- Still plays legendary reveals (abbreviated to 1 second)
- All other items appear instantly

**Auto-Skip After 15 Seconds:**
- If player walks away, automatically complete reveal
- Prevents softlock

## Audio Specifications

### Sound Categories

**Gold Sounds:**
- Coin drop: Metallic clinks (3 variations to avoid repetition)
- Counter tick: Digital beep, increasing pitch (10-15 pitch levels)
- Lock-in: Deep satisfying "ka-chunk" with bass

**Gem Sounds:**
- Spawn: Crystalline chime (higher pitch than gold)
- Counter tick: Glass bell tones (musical scale)
- Lock-in: Bright "shine" sound with reverb

**Chest Sounds:**
- Drop: Heavy wooden thud + screen rumble
- Shake: Creaking wood
- Open: Hinge squeak + burst of magical sparkle

**Slot Machine Sounds:**
- Spin start: Mechanical whir (looping)
- Ticking: Rhythmic clicks (tempo decreases with speed)
- Stop common: Simple "thunk"
- Stop rare: "Thunk" + crystalline accent
- Stop legendary: "CHUNK" + choir/orchestra hit + echo

**Legendary Reveal:**
- Screen freeze: Time-stop "whoosh"
- Explosion: Bass boom + high sparkle
- Rotation: Heavenly angelic choir (1-2 seconds)
- Final lock: Triumphant horn fanfare

### Audio Mixing Priority

1. Legendary sounds (100% volume, mute/duck everything else)
2. Slot machine results (80% volume)
3. Counter ticks (60% volume, layered)
4. Particle sounds (40% volume, subtle)
5. Background music (20% volume during reveals, fades back to 60% after)

## Performance Optimization

### Particle Budget per Phase

- Gold phase: Max 12 particle tiles
- Gem phase: Max 16 particle tiles
- Chest phase: Max 8 particle tiles
- Each slot reveal: Max 10 particle tiles (50 total if 5 slots)
- Legendary reveal: Max 60 particle tiles (unlimited budget, special case)

### Framerate Targets

- Normal animations: 60 FPS
- Legendary slow-mo: 30 FPS (intentional)
- Fast-forward: 120 FPS (speed up animations)

### Memory Considerations

- Reuse tile sprites (don't create new objects)
- Pool particle tiles (max 100 particle objects alive at once)
- Clear particles after they fade (free memory)
- Cache sound effects (don't reload each time)
